# Meu Site Pessoal

## Descrição

Um site para me apresentar, mostrar meus projetos e facilitar o contato. Este arquivo serve como um guia e planejamento para o desenvolvimento do site.

## 🎯 Planejamento e Funcionalidades

Aqui está uma lista das seções e funcionalidades planejadas para o site. Marque os itens conforme forem concluídos.

- [ ] **Seção Sobre Mim:** Uma introdução mais detalhada sobre quem eu sou, minhas paixões e minha carreira.
- [ ] **Seção de Projetos/Portfólio:** Uma galeria ou lista com os projetos mais importantes que desenvolvi.
- [ ] **Formulário de Contato:** Uma maneira para os visitantes entrarem em contato comigo diretamente pelo site.
- [ ] **Links para Redes Sociais:** Ícones e links para meu LinkedIn, GitHub, etc. (Já iniciado!)
- [ ] **Design Responsivo:** Garantir que o site funcione bem em celulares e tablets.

## 💻 Tecnologias Utilizadas

*   HTML5
*   CSS3
*   JavaScript

## 🚀 Como Rodar o Projeto Localmente

1.  Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2.  Abra o terminal na raiz do projeto.
3.  Execute o comando abaixo para iniciar um servidor local:
    ```bash
    npx http-server .
    ```
4.  Abra seu navegador e acesse `http://localhost:8080`.
